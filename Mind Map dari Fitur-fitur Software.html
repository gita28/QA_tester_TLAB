Mind Map of Software Features

  • 1. Sub Sistem Perangkat IoT

    • Sensor Lingkungan

      • Sensor Suhu Lingkungan
      • Sensor Kelembaban Lingkungan
      • Sensor Intensitas Cahaya Lingkungan
      • Pengambilan Gambar Kondisi Lingkungan

    • Daya

      • Pengisian Daya dari Solar Panel

  • 2. Sub Sistem Komunikasi

    • Modem 4G

      • Pengiriman Data Sensor:
        • Data Suhu Lingkungan
        • Data Intensitas Cahaya
        • Data Kelembaban Udara
        • Data Gambar Kondisi Perangkat Serangga
      • Pengiriman Data Perangkat:
        • Nomor Pengenal Perangkat IoT
        • Status Aktif/Non-Aktif Perangkat IoT

      • Penerimaan Perintah

        • Aktivasi Sensor dari Back End

      • Protokol Komunikasi

        • MQTT
        • HTTP

  • 3. Sub Sistem Back End

    • Penerimaan dan Penyimpanan Data Sensor

      • Data Suhu Lingkungan
      • Data Intensitas Cahaya Lingkungan
      • Data Kelembaban Udara
      • Data Gambar Kondisi Perangkat Serangga

    • Penerimaan dan Penyimpanan Data Perangkat

      • Nomor Pengenal Perangkat IoT
      • Status Aktif/Non-Aktif Perangkat IoT

    • Pengiriman Data Sensor ke Front End

      • Data Suhu Lingkungan
      • Data Intensitas Cahaya Lingkungan
      • Data Kelembaban Udara
      • Data Gambar Kondisi Perangkat Serangga

    • Penyimpanan

      • Gambar ke Object Storage
      • Data ke Database Postgres

  • 4. Sub Sistem Front End

    • Menampilkan Data Sensor

      • Suhu Lingkungan
      • Intensitas Cahaya Lingkungan
      • Kelembaban Udara
      • Gambar Kondisi Perangkat Serangga

    • Menampilkan Grafik Hasil Sensor

      • Berdasarkan Jangka Waktu (Harian, Mingguan, Jam)
      • Berdasarkan Lokasi Perangkat (Peta)
      • Berdasarkan Identitas Perangkat IoT

    • Notifikasi

      • Status Aktif/Non-Aktif Perangkat IoT
      • Data Baru dari Sensor Lingkungan

    • Pengiriman Data ke Back End

      • Aktivasi Sensor Lingkungan
      • Pengaturan Jadwal Pengiriman Data

    • Pengelolaan Data

      • Tambah/Edit/Hapus Data Perangkat IoT
      • Tambah/Edit/Hapus Foto Koleksi Serangga

    • Hak Akses

      • Role-Based Access Control
